🍎 MacBook Clean-Up Guide

Disk Space · File Organization · Security & Privacy

💾 1 — Free Up Disk Space

🚀 Quickest Win: Run the Cleanup Script

Clone the repo and run in Terminal:

# Clone and run
git clone https://github.com/PRGOLD/macbook-cleaner.git
cd macbook-cleaner && chmod +x mac_cleanup.sh && ./mac_cleanup.sh

🍎 Built-in macOS Storage Management

  1. Go to Apple menu → System Settings → General → Storage
  2. Click ⓘ next to categories to see what's taking space
  3. Use Optimize Storage, Empty Trash Automatically, and review Large Files

📦 Common Space Hogs

LocationTypical Size
~/Downloads1–20 GB
~/Library/Caches1–10 GB
Xcode DerivedData5–50 GB
Docker images5–30 GB
~/Movies / ~/Photos10–100 GB+
Old iOS backups5–20 GB

🔍 Find Big Files via Terminal

Largest items in your home folder:

du -sh ~/* 2>/dev/null | sort -rh | head -20

Files larger than 500 MB:

find ~ -size +500M -not -path "*/.*" 2>/dev/null

📱 Remove Old iPhone/iPad Backups

  1. Open Finder → click your device in the sidebar → Manage Backups…
  2. Delete backups you no longer need
💡 If you use iCloud Backup, you may not need local backups at all.

🔒 3 — Security & Privacy

🛡️ Security Checklist

🔍 Check Login Items via Terminal

# User LaunchAgents
ls -la ~/Library/LaunchAgents/
# System-wide LaunchAgents
ls -la /Library/LaunchAgents/
⚠️ Don't delete LaunchAgents unless you're sure what they are — some are required by legitimate apps.
⚡ Bonus — Speed & Battery

🐢 Mac Running Slow?

  • Open Activity Monitor → check CPU and Memory tabs for resource hogs
  • Reduce Login Items (see Security section above)
  • Restart your Mac to clear RAM

🔋 Battery Health

  • System Settings → Battery → Battery Health → Optimized Battery Charging
  • Hold ⌥ → Apple Menu → System Information → Power to check cycle count
